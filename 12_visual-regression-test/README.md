## 課題2
スナップショットテスト  
- 出力されるコードに変化がないか把握する際に有用。画像のalt属性やaタグのリンク先の変更も検知できる。

ビジュアルリグレッションテスト  
- HTML/CSSを修正した際に起こる見た目の変化の影響範囲を正確に把握できる。
- 出力されるソースコードに変化があっても見た目さえ変わっていなければテストが通るので、
タグ名やクラス名のリファクタリングをする際などに使いやすい。
