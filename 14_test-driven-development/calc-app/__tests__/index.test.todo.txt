add
[] 引数の数字を足して返す
  [] 引数に1と2を渡したら3が返ってくる
  [] 引数に1と2と3を渡したら6が返ってくる
[] 計算結果が1000を超える場合、too bigと文字列が返る
  [] 引数に1000と1を渡すとtoo bigが返る
  [] 引数に1000と0.001を渡すとtoo bigが返る

substract
[] 引数の数字を引いて返す
  [] 引数に1を渡したら1が返ってくる
  [] 引数に2と1を渡したら1が返ってくる
  [] 引数に2と-1を渡したら3が返ってくる
[] 計算結果がマイナスの場合、negativeと文字列が返る
  [] 引数に1と2を渡すとnegative numberと文字列が返る
  [] 引数に0, 0.00000001を渡すとnegativeと文字列が返る

common
[] 1~30個までの引数を受け取ることができる。
　[] 引数が1*30個で30を返す
[] 引数が31個以上でエラーを発生させる
  [] 引数が31個でエラーが発生する
[] 引数が数字意外だとエラーが発生する
  [] 文字列"1"を渡すとエラーが発生する
  [] 配列[1,2]を渡すとエラーが発生する

[] 掛け算の場合、計算結果が1000を超える場合はbig big numberと文字列が返る　など
[] 割り算の場合、 結果は小数第5位で四捨五入し、小数第4位まで表示　など
